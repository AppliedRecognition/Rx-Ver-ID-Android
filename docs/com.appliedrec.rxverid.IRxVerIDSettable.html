IRxVerIDSettable

Interface that provides a method to set an instance of com.appliedrec.rxverid.RxVerID.html

If implemented in an activity and the activity's application has an instance of RxVerID registered using com.appliedrec.rxverid.RxVerIDHelper.html, the activity will automatically have an instance of RxVerID set in android.app.Activity.html#onCreate(Bundle) and removed in android.app.Activity.html#onDestroy().
